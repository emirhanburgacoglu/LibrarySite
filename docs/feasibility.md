# Feasibility Study

## Technical Feasibility
The project will be developed using C# and ASP.NET MVC.
The team has sufficient background to implement a web-based library management system.

## Operational Feasibility
The system is designed for university students and library staff.
It simplifies book and study room reservation processes.

## Schedule Feasibility
The project will be developed incrementally using the Kanban methodology.
Tasks are divided into small units to ensure timely completion.
